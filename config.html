Quick Setup

  1. Start the local server: python server.py in the project folder
  2. Enter your Tunarr server URL and click Auto-fill
  3. Adjust options, then click Save & Open Guide

Tunarr must have CORS enabled (it does by default). XMLTV and M3U URLs are auto-filled from your server URL — only change them if you're using external guide sources.
